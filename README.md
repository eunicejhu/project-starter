## How to setup project-starter?

- init project
- install webpack
- create webapack config files
- setup file structure
- setup babel, browserslist
- setup style loaders
- setup file-loader

```sh
npm init
npm i -D webpack webpack-cli
mkdir public src
touch webpack.base.js webpack.dev.js webpack.prod.js
```
